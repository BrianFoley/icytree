phylo.js

Javascript library for phylogenetic tree manipulation and visualisation.

View project onGitHub

This is a compact JavaScript library for parsing Newick representations of phylogenetic trees and visualizing the result using SVG. It is currently composed of two parts:

  1. phylo.parse.js which defines the prototype node and tree objects and contains methods for constructing trees by parsing Newick-formatted strings (and NEXUS files containing these strings).
  2. phylo.vis.js which defines the prototype layout object which allows tree objects to be given a spatial representation and posesses methods to generate SVG visualisations of these representations.
